# Blog App

## Requirements
- user can post and publish blog content
- user can see post
- authentication system
- user can see their own profile

## Table
- Post
    - id
    - title
    - content
    - authorId
    - published
    - createdAt
    - updatedAt

- User
    - id
    - name
    - email
    - password
    - profile
    - createdAt
    - updatedAt

- Profile
    - id
    - bio
    - userId
    - createdAt
    - updatedAt

## Technology Stack:
    - graphql
    - typescript
    - postgresql
    - prisma
